# terrahub list

```text
Usage: terrahub list [options]

terrahub@0.0.1 (built: 2018-04-14T19:15:39.787Z)
this command will list cloud resources by projects > accounts > regions > services > resources

Options:
  --depth, -d       Listing depth (0 - projects, 1 - accounts, 2 - regions, 3 - services, 4 - resources)
  --projects, -p    Projects (comma separated values)
  --accounts, -a    Accounts (comma separated values)
  --regions, -r     Regions (comma separated values)
  --services, -s    Services (comma separated values)
  --env, -e         Workspace environment
  --help, -h        Show list of available commands
```
